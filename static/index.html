Connect MetaMask

This demo lets a user connect their MetaMask wallet and (optionally) link it to a user key via signature.

If you plan to use a bot: set this to the Telegram user id (or pass it as ?user_key=...)
Status: Not connected
Address: —
What does “link” do?

It requests a nonce from the server, asks MetaMask to sign a message, then the server verifies the signature and stores user_key → wallet_address in SQLite.
